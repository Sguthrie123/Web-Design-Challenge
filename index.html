Latitude X


The purpose of this project was to analyze the weather and how it changes due to the latitude of your coordinates. I specfically chose the range latitude of Europe because that is where I wanted to go on vacation before the quarantine happened. To do this we had to use the OpenWeatherMap API to pull data of those cities near the latitude we chose.
We then uses MatPlotLib to plot various aspects of the data we recieved from the API. Some factors we looked at were the temperature, cloudiness, wind speed, humidity. If you look at the comparisons, you will see all the visualization of the data we found.
